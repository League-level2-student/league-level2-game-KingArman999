Box Dodge

by: Arman Dahi

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because I enjoy dodger games. I couldn't think of any other game to make so I settled on my first idea.

2. What I Learned

I learned that while many people enjoyed my game, there were still things to fix and improve on. I got feedback that caused me to make it so that all of the numbers line up on the high score list.

3. Architecture of My Game

There are five parts to this game: the menu, the instruction, the game, the fail, and the high score screens. The menu, instruction, fail, and high score screens should be pretty self explanatory. The game screen has a user controlled dodger and falling boxes that the user must dodge in order to progress.
